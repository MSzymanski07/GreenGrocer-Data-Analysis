# GreenGrocer-Data-Analysis (Module 12 Assignment)
### The Purpose of this Program
This program is designed to analyze and visualize the business data of a fictional firm with the goal of provding the firm with useful results that inform future decisions.
### Overview of Program Uses
1. Provides current performance data
2. Diagnostic Analysis
3. Predictive Analysis/Basic Forecasting
4. Provides Business Insights and Recommendations
### Notes to Mention
The data used is randomly generated but is reproducible with other data sets.
The code itself returns several matplotlib figures generated from data and calculations.
Requires installation of pandas, numpy, matplotlib, and scipy.
Designed as part of an educational assignment, but is functional with real data sets.
### Deeper Look at Program
1. Analyzes sales performance
2. Visualizes sales distribution
3. Analyzes customer segments
4. Analyzes sales correlations
5. Compares location/store performance
6. Analyzes seasonal patterns
7. Predicts location/store sales
8. Forecasts sales in specific departments/divisions
9. Identifies profit opportunities
10. Develops business recommendations
11. Generates an executive summary for the user
